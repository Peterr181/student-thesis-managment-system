Program to manage student thesis and modify them!

It's very simple program created for managing, editing and more student thesis based on
c++ objected oriented programming. It was a project to get final grade of c++ programming subject in the college.

Features:
-Displaying the job list in a shortened format
-Displaying the job list in a full format
-Adding works to library
-Edit data about work
-Find work depend on option
-Additional functions 
-- Saving data of all added works to binary file and store this in them
